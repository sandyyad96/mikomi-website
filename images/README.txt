Drop optional raster images here (student photos, founder photo, logo PNG, etc.).

The site currently uses crisp inline SVG artwork and emoji icons, so no raster
images are required for the page to look complete. To use a real founder photo,
replace the ".founder-avatar" block in index.html with an <img> tag pointing
to a file in this folder, e.g.:

  <img src="images/astha.jpg" alt="Astha Pokhriyal, Founder of Mikomi" />
